# AMM v1 API Specification

This directory contains formal JSON schemas for the AMM v1 API.

## Files

- `mcp.tools.v1.json` - MCP tool definitions schema
- `cli.command.schema.json` - CLI command envelope schema
- `payloads.schema.json` - Request/response payload schemas

## Version

Version: `amm.v1`

## Schema Usage

These schemas are used for:
1. Request/response validation in tests
2. Documentation generation
3. Client code generation
4. Contract compatibility checking
